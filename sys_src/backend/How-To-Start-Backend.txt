Evtl. zuerst virtualle Environment installieren
    pip install virtualenv

1. Virtuelle Umgebung starten:
    source bin/activate

2. (nur das erste mal) requirements importieren:
    pip install -r requirements.txt

3. Webserver starten (ip steht dann in der Konsole):
    python -m uvicorn main:app --reload

4. UnitTest mit Coverage (bei anderen File als main, einfach file anstelle von main einfÃ¼gen):
    python -m pytest --cov main --cov-branch --cov-report term-missing

5. Sollten neue Packages installiert werden
    pip freeze > requirements.txt
